Adds a little arrow on top of your fields, which allows you to collapse them. Works in the Current Card Editor, Browser, and the Add Cards dialogs. Shortcuts All shortcuts are settable through "Tools > Add-ons > Collapsible Fields > Config".
  • F10: Toggle Collapsible status for current field
  • Shift+F10: Uncollapse / Collapse all fields which are collapsed by default and non-empty
Collapse by default You can set fields to collapse by default. To enable this, open the editor, select "Fields", choose your field, and check "Collapse by default".
